# Project Archive Template

## Archive Location & Naming

**Location:** `.claude/project-coordinator/archives/`
**Naming:** `[topic]_[YYYYMMDD].md` (e.g., `ci-test-failure_20250115.md`)

## Template

```markdown
# [Project Title]

**Completed:** YYYY-MM-DD

## Purpose (from purpose.md)

[Original objective and success criteria - copy verbatim]

## Approach Summary (from plan.md)

### Final Plan
[Key steps that were executed]

### What Worked
- [Successful approaches]

### What Didn't Work
- [Failed attempts and why - important for future reference]

## Key Findings (from research_memo.md)

[Condensed insights - focus on discoveries that have lasting value]

- **Finding 1:** [Description]
- **Finding 2:** [Description]

## Outcome

[Was the purpose achieved? Any caveats or remaining considerations?]

## Future Reference

[Anything useful for similar future tasks - commands, configurations, gotchas]
```

## Guidelines

- Keep concise - this is a summary, not a full copy
- Focus on learnings over chronological details
- **Always include failed attempts** - they prevent future repetition
- Link to any relevant code/docs created during the project
