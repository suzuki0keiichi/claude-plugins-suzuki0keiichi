# Project Archive Template

Use this template when creating project archives at completion.

## Archive File

**Location:** `.claude/project-coordinator/archives/`
**Naming:** `[topic]_[YYYYMMDD].md` (e.g., `ci-test-failure_20250115.md`)

## Template

```markdown
# [Project Title]

**Completed:** [YYYY-MM-DD]
**Duration:** [Start date] - [End date]

## Purpose (What & Why)

[Copy from purpose.md - original objective and success criteria]

## Approach (How)

[Summary from plan.md - key steps taken, major decisions]

### What Worked
- [Successful approaches]

### What Didn't Work
- [Failed attempts and why]

## Key Findings

[Condensed insights from research_memo.md]

- **Finding 1:** [Description]
- **Finding 2:** [Description]

## Outcome

[Final result - was the purpose achieved? Any caveats?]

## Future Reference

[Anything useful for similar future tasks]
```

## Guidelines

- Keep concise - this is a summary, not a copy
- Focus on learnings over chronological details
- Include failed attempts - they prevent future repetition
- Link to any relevant code/docs created
